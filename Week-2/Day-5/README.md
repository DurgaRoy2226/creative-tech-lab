\# Week 2 – Day 5: Git \& GitHub Practice



\## Task 1 – Initialize Portfolio Repo

\- Open your existing Personal Portfolio Project folder

\- Run:

&nbsp; - git init

&nbsp; - git add .

&nbsp; - git commit -m "Initial Portfolio Commit"

&nbsp; - git branch -M main

&nbsp; - git remote add origin https://github.com/yourusername/portfolio.git

&nbsp; - git push -u origin main



\## Task 2 – Invite a Peer

\- Go to GitHub → Settings → Collaborators

\- Add your classmate’s GitHub username



\## Task 3 – Create and Merge a Feature Branch

1\. Create a new branch:

&nbsp;  - git checkout -b feature-projects

2\. Add new section in HTML/CSS

3\. Commit \& push:

&nbsp;  - git add .

&nbsp;  - git commit -m "Added projects section"

&nbsp;  - git push origin feature-projects

4\. Create Pull Request on GitHub and merge into main
